转换成字符串

  • 调用 toString() 方法 （显式转换）

null 、undefined 没有 toString() 方法

  • 调用 String() 构造函数（显式转换）
  • 直接和字符串做加法运算（隐式转换）
  • 引用类型转换成字符串

转换成数值

  • Number()
  • 使用 parseInt()
  • 使用 parseFloat() 完成
  • 让字符串和数字做除了加法以外的运算（隐式转换）
  • 引用类型转换成数值

转换成布尔值

0, NaN, 空字符串，undefined, null, false 会被转换成为 false

  • 使用 Boolean() 完成
  • 使用 !!
  • 自动转换
