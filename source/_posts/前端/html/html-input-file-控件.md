## 清空 value

使用 input file 进行文件上传时，重复选择相同文件时，change 事件不再触发
解决方式：手动将 file 的 value 值设置为空

```html
<input type="file" id="file" accept="image/gif,image/jpeg" @change="uploadFile($event)" />
```

```js
$event.target.value = ''
// 或
var file = document.getElementById('file')
file.value = ''
```
