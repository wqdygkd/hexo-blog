## 快速删除 node_modules 文件夹

系统命令

```bash
# cmd
# /q 直接删除，不需要二次确认
# /s 允许删除非空目录
rd /q/s node_modules

# powershell
rmdir -r -Force node_modules
# 或
rm -r -Force node_modules
```

使用 npm 包

```bash
npm i rimraf -g
rimraf node_modules
```

## 一次执行多条命令

```bash
# cmd 使用 && 连接多个命令
cd vue && npm i
# powershell 使用 | 连接多条命令
cd vue | npm i
```


## 删除一个文件夹内字节小于 10kb 的文件

```bash
# 输出所有大小小于10k的文件
# find ./ -size -10k > NoNeedSite.txt
find ./ -size -10k -exec rm {} \
```
