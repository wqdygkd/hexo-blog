修改 host 方式

访问 http://tool.chinaz.com/dns 或 https://www.ipaddress.com/

查询以下三个链接的 DNS 解析地址

找到 TTL 值最小的 IP，修改系统 Hosts 文件

之后刷新系统 DNS 缓存

Windows+X 打开系统命令行（管理员身份）或 powershell

运行 ipconfig /flushdns 手动刷新系统 DNS 缓存。
